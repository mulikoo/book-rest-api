# Book REST API

## Описание проекта

Book REST API — это простое и функциональное веб-приложение, представляющее собой backend-часть для управления коллекцией книг. Оно реализует архитектурный стиль REST и предоставляет стандартный набор операций CRUD (Create, Read, Update, Delete).

**Для кого предназначен:**
*   Для начинающих разработчиков, изучающих Spring Boot и создание RESTful сервисов.
*   В качестве основы для более сложных проектов, требующих функционала работы с книгами или другими сущностями.

**Основные функции:**
*   **Просмотр каталога:** Получение списка всех книг или информации о конкретной книге по её ID.
*   **Добавление новых книг:** Создание новых записей о книгах в коллекции.
*   **Редактирование информации:** Обновление данных о существующей книге.
*   **Удаление книг:** Удаление книг из коллекции.

## Установка и запуск

Для запуска проекта локально вам потребуется:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/mulikoo/book-rest-api.git
    cd book-rest-api
    ```

2.  **Убедитесь, что у вас установлены:**
    *   **Java 21** (или более поздняя версия).
    *   **Maven** (обычно используется обёртка `mvnw`, которая есть в проекте).

3.  **Запустите приложение:** Вы можете сделать это одним из двух способов:
    *   С помощью Maven-обёртки (рекомендуется):
      ```bash
      ./mvnw spring-boot:run
      ```
    *   Или через ваш IDE: найдите класс `BookApi2Application` и запустите метод `main`.

4.  **Проверьте запуск:** После успешного запуска в консоли вы увидите логи Spring. Приложение будет доступно по адресу: `http://localhost:8080`.
